晶片操作

按下快捷鍵 “；” 開啟晶片介面

  1. 將 83等以上裝備 右鍵至晶片介面中

  2. 按下 “欄位顏色” 開啟晶片格

  3. 當顏色不對時，可以使用 強化槽類型變更工具 改變格子顏色
    搭配驅動器 或 真實驅動器，分別有 提高該顏色機率 或 必定獲得該顏色 的效果

湊到相應數量的材料後，在右側點想做晶片，再按 “製作” 即可製作晶片

用於在裝備上面 安裝或解除晶片 的功能
點已開啟的格子，在右方選擇晶片即可安裝

將晶片能力進一步強化
點已安裝的晶片，在右方選擇晶片即可超頻
若選擇相同顏色的材料晶片，則提供的經驗值為 2 倍

將晶片Tier數提升並保留超頻總經驗值
點已安裝的晶片，在右方選擇晶片即可繼承
繼承會保留先前晶片超頻的總經驗值，沒有繼承懲罰

將晶片從舊裝備轉移到新裝備上並保留超頻總經驗值
點已安裝的晶片，在右方選擇晶片即可繼承
轉移需要一定數量的 晶片保護裝置 (可於榮譽每周購買一個)
舊裝備先轉移強化時，舊裝備上的晶片超頻等級仍會保留
所以在換裝備的時候，步驟如下：

  1. 先轉移強化
  2. 再轉移晶片

超頻經驗表

  • 超頻等級N：指 N-1 到 N 等經驗，例如N = 13就是 12 到 13 等的超頻經驗值
  • 雖然強化上限為+30，但是目前晶片超頻上限仍為+25
  • 也可透過 線上Excel 或 離線檔案 查看更詳細的內容，裡面包含超頻總經驗值的數據。

建議晶片配置

  • 混合角色以提升條件爆傷為主，可最大限度同時提升物理與魔法的傷害
    黑色替換：緹娜使用白色，納塔可使用紫色代替

  • 若無 T1鉑金晶片 可用 早期鉑金晶片 替代
  • 武器： 最優先 ，盡量用最高T值(T3 T4)
  • 模組：次優先
  • 護盾：最低考量，顏色晶片可使用介於T2~T3之間，比T3便宜許多的 伯爾戴布系列
  • 接收器：除非要上 T4鉑金 不然建議使用 提亞瑪特(女王)系列
  • 額外參考：傷害公式與暴抗表。主要探討武器的 紅色 與 藍色 位置的選擇(可使用 黑色 或 白色 或 紫色 )
