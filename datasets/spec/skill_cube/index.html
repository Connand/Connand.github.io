技能核心使用介面

使用介面

技能核心將會變成帳號制，有專屬背包可以存放

(打勾處)顯示自己類型
(黃色按鈕)核心合成
高級 菁英
傳說 覺醒


可以放置2種核心：原版的(傷害增加等等) + 覺醒技能

核心合成

使用介面

使用3個相同等級(例如傳說或覺醒等)的技能核心可以進行合成，10%機率可以合成到高一階的核心

目前取得技能核心的管道

此系統開放後，取得技能核心箱子的管道將會受約束(尤其是製作機與販賣機)

  1. PVP商店
  2. 貢獻度(榮譽)商店
  3. 平原之門製作機->特殊作戰區域
  4. 平原之門副本(紅藍本)

覺醒技能核心

此系統更新除了將原本大淨化作戰(雷本)的覺醒技能改為不受裝備制約。
並每個角色都新增了一招 新覺醒技能 ，變更技能為獎章技能(俗稱鎖技)

取得來源

新舊覺醒技能核心取得方式如下 請注意新舊覺醒道具圖外框四角的差別

  • 舊覺醒技能核心(雷本覺醒)
    1. （已無法製作）平原之門深層製作機：可製作各一個自身角色的覺醒技能。 製作限制：每個角色1次
    2. 使用覺醒技能核心合成(90%機率)
    3. 使用傳說或菁英核心合成(10%機率)
    4. 未鑑定覺醒技能核心(不可交易，90%機率 **軍團程式獎勵（請參考下方）** )
  • 新覺醒技能核心
    1. 使用覺醒技能核心合成(10%機率)
    2. 未鑑定覺醒技能核心(可交易，10%機率 **軍團程式獎勵（請參考下方）** )

軍團程式

軍團程式(或軍團計畫，Legion Program)，旨在考驗種族增傷的輸出能力檢測(主要為突變PNA)
目前開放的副本分為五種：2023/04/20新增特異型、2023/05/03新增業火型

食蟲型 海底型 前腦型 野生型 特異型 業火型

入場與玩法

舊版

入場建議戰力為1000萬。共分 10 個 WAVE，每個 WAVE 通常需要擊殺 30 隻怪物即可通關此 WAVE。

新版

  • 韓版 2025/01/23 改版內容
    入場建議戰力為1000萬。但改為無限出怪。
    擊殺怪物可獲得積分，每 1000 分可獲的獎勵，最多 30000 分。
    並且會有一個副本技能，能量條滿時使用(Ctrl + Q)會恢復 100% MP，並且重置終極技以外所有技能的冷卻。

樓層獎勵

  • 每周帳號入場 2 次 (6 種副本共用入場次數，可重複入場)
  • 若滿分可得：x1040 x77 x24 x6
  • ：使用後可獲得 PNA 經驗 5,000。
