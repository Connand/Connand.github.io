強化與晶片機率表

發表於2021-09-03|更新於2024-09-15|遊戲資料機率資料
|閱讀量:

來源為韓版官方公布的機率表：https://closers.nexon.com/News/Guide/List?n4ArticleCategorySN=4#today

公告中提到：

  • 所有機率均已通過遊戲記錄分析驗證，可正常運行。
  • 因為四捨五入到小數第二位會造成些許差異，總機率應為100%，但根據此表計算總機率不一定是100%。
  • ％ 標示到小數點第二位，比例到小數點第一位。(不清楚意思)

使用一般強化器燃料時的機率表

注意：

  • 使用幸運強化器燃料時，成功率為1.3倍
  • +20機率表：此為韓版剛公布機率表時所記錄的。
  • +22機率表：此為韓版開放機械王同時開放+22所修改的表，0-10的各機率不變
  • +25機率表：因應韓版2022/07/28開放野獸王並將強化上限提升至+25所修改的機率表
  • +30機率表：因應台版2024/09/12開放魔龍王並將強化上限提升至+30所修改的機率表
  • 每個裝備的 保證強化點數 各自獨立
  • 使用 一般強化器燃料 強化結果為無變化時， 保證強化點數 增加 100 點
  • 使用 幸運強化器燃料 強化結果為無變化時， 保證強化點數 增加 130 點
  • 強化失敗後僅能保留對應等級的 保證強化點數 上限，超出的點數將被移除
  • 當 保證強化點數 到達對應上限時，下次強化 必定成功 並重置點數為 0 點
  • 一般保底 / 幸運保底 表示只使用 一般強化器燃料 / 幸運強化器燃料 達成保證強化所需的強化次數
